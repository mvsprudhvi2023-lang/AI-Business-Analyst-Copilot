# AI Business Analyst Copilot

AI Business Analyst Copilot is an AI-powered business analytics dashboard that helps users analyze sales data through automated KPIs, SQL analytics, visualizations, business insights, machine learning predictions, and downloadable reports.

## Features

- Upload sales CSV files
- Preview dataset rows and columns
- View summary statistics
- Generate KPI metrics
- Analyze sales by region
- Analyze profit by category
- Run SQL-based sales analysis
- Generate AI-style business recommendations
- Predict next sales value using Linear Regression
- Download a business report

## Tech Stack

- Python
- Pandas
- NumPy
- Streamlit
- SQLite
- Scikit-Learn
